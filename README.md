# DukeCaMa

This repository contains DukeCaMa (Duke Capstone Ops), a role-based portal for managing capstone projects across students, faculty, admins, and sponsors.

## Structure

- frontend/ - React + TypeScript + Vite app

## What it does

- Role-based dashboards for Admin, Faculty, Student, Client, and Guest views
- Student workspace with project hub, task board, repository/code view, team health, and chat
- Faculty and admin views for oversight, teams, users, and settings
- Client sponsor dashboard for reviewing deliverables
- GitHub repo insights (commits, pull requests, issues, contributors)

## Getting started

```bash
cd frontend
npm install
npm run dev
```
